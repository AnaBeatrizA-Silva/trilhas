# Repositório de Atividades
Este repositório contém as trilhas de atividades das trilhas

## Como utilizar
1. Faça um **Fork** deste repositório para sua conta do GitHub
2. Clone o seu fork para sua máquina

```bash
git clone https://github.com/user/repo.git
```

3. Desenvolva as atividades dentro das respectivas trilhas
4. Faça commits conforme o desenvolvimento

> **Importante:** Não altere a estrutura de pastas do projeto

## Estrutura

```text
repo/
│
├── trilha_sql/
│   └── ...
│
└── trilha_poo/
    └── ...
```

Cada trilha contém atividades independentes

- **trilha_sql** → relacionados a SQL
- **trilha_poo** → relacionados à POO
